Operasi pada Himpunan dan Sifat Operasi Antar Himpunan

  1. Gabungan Dua Himpunan

    1. Pengertian dua himpunan

      2. Jika A dan B adalah dua himpunan, gabungan himpunan A dan B adalah himpunan yang anggota - anggotanya terdiri atas anggota A atau anggota B. Gabungan antara dua himpunan dinotasikan denagn A ⋃ B, dan dapat ditulis dengan A ⋃ B = {x | x ∈ A atau x ∈ B}

    2. Menentukan gabungan dua himpunan

      3. - Himpunan yang satu merupakan himpunan bagian dari himpunan yang lain. Jika A ⊂ B maka A ⋃ B = A

      - Kedua himpunan sama. Jika A = B maka A ⋃ B = A = B

      - Kedua himpunan tidak saling lepas(berpotongan)

    3. Menentukan banyaknya anggota dari gabungan dua himpunan

      4. Dirumuskan dengan n(A ⋃ B) = n(A) + n(B) - n(A ⋃ B)

  2. Irisan Dua Himpunan

    1. Pengertian irisan dua himpunan

      2. Irisan (interaksi) dua himpunan adalah sutau himpunan yang anggotanya merupakan anggota persekutuan dari dua himpunan tersebut. Irisan himpunan A dan B dinotasikan sebagai berikut . A ⋂ B = {x | x ∈ A atau x ∈ B}

    2. Menentukan irisan dua himpunan

      3. - Himpunan yang satu merupakan himpunan bagian dari himpunan yang lain. Jika A ⊂ B maka A ⋂ B = A

      - Kedua himpunan sama. Jika A = B maka A ⋂ B = A atau A ⋂ B = B

      - Kedua himpunan tidak saling lepas(berpotongan)

  3. Selisih (Difference) Dua Himpunan

    4. Selisih (difference) himpunan A dan B adalah himpunan yang anggotanya semua anggota dari A tetapi bukan anggota dari B. Selisih himpunan A dan B dinotasikan dengan A - B atau A\B. Notasi pembentuk himpunannya dituliskan sebagai berikut.

    A - B = {x | x ∈ A, x ∉ B}

    B - A = {x | x ∈ B, x ∉ A}

  4. Komplemen Suatu Himpunan

    5. Komplemen himpunan A adalah suatu himpunan yang anggota - anggotanya merupakan anggota S tetapi bukan anggota A. Komplemen dinotasikan dengan A^C atau A', dibaca komplemen himpunan A. Dengan notasi pembentuk himpunan ditulisakn seperti di bawah ini:

    AC = { x | x ∈ B, x ∉ A }
  5. Sifat - Sifat Operasi Himpunan

    1. Sifat - sifat irisan dan gabungan himpunan

      2. - Sifat Komutatif irisan A ⋂ B = B ⋂ A

      - Sifat Asosiatif irisan (A ⋂ B) ⋂ C = A ⋂ (B ⋂ C)

      - Sifat idempotent irisan

    2. Sifat - sifat selisih himpunan

      3. - Sifat distributif selisih terhadap irisan. Untuk setiap himpunan A, B, dan C berlaku A – (B ⋂ C) = (A – B) ⋃ (A – C)

      - Sifat distributif selisih terhadap gabungan A – (B ⋃ C) = (A – B) ⋂ (A – C)
