LATIHAN BAB V

  1. Perhatikan Diangran Venn berikut !

    2. Diketahui S = {siswa yang gemar olahraga}; P = {siswa yang gemar bola voli}; Q = {siswa yang gemar bola basket}; Setiap anggota himpunan ditunjukkan dengan noktah. Dari diagram Venn tersebut, sebutkan anggota himpunan berikut.

    1. Himpunan siswa yang gemar olahraga
    2. Himpunan siswa yang gemar bola basket
    3. Himpunan siswa yang gemar bola voli dan basket
    4. Himpunan siswa yang gemar bola voli saja
    5. Himpunan siswa yang gemar bola basket saja

  2. Dari diagram Venn di bawah, tentukan

    1. anggota himpunan S;
    2. anggota himpunan A;
    3. anggota himpunan B;
    4. anggota himpunan C

  3. Berdasarkan diagram Venn pada soal nomor 2 di atas, tentukan

    1. anggota himpunan A ⋂ B
    2. anggota himpunan A ⋃ B
    3. anggota himpunan B ⋂ C
    4. anggota himpunan A ⋂ B ⋂ C
    5. anggota himpunan A ⋂ Bc
    6. anggota himpunan B\C

  4. Diketahui himpunan-himpunan berikut S = {bilangan cacah kurang dari 15}; A = {lima bilangan ganjil yang pertama}; B = {lima bilangan genap yang pertama}; C = {faktor dari 8}; D = {tiga bilangan kuadrat yang pertama}

    1. Nyatakan himpunan-himpunan di atas dengan mendaftar anggotanya
    2. Buatlah diagram Venn untuk masing-masing himpunan berikut, dengan S sebagai himpunan semestanya

    1. Himpunan S, A, dan B
    2. Himpunan S, A, dan C
    3. Himpunan S, B, dan D
    4. Himpunan S, A, C, dan D
    5. Himpunan S, B, C, dan D

  5. Arsirlah daerah yang menggambarkan A ⋂ B untuk setiap himpunan yang disajikan oleh diagram Venn berikut

  6. Dari diagram Venn di bawah, tentukan

    1. n (S);
    2. n (A);
    3. n (B);
    4. n (A ∩ B)
    5. n (A ∪ B)
    6. n (A c);
    7. n A/B
    8. n (A ∩ B)c;
