LATIHAN BAB IV

  1. Bila diketahui P = {1,2 , 4, 7, 11} dan Q = {2, 3, 5, 7 ,11}, tentukan P ⋂ Q !
  2. Tentukan P ⋂ Q dengan menyebutkan anggota-anggotanya, kemudian tentukan n(P ⋂ Q) untuk himpunan P dan Q di bawah ini.

    1. P = {x | 0 < x d 5, x ∈ A}
      2. Q = {x | –4 d x < 1, x ∈ B}
    2. P = {x | x < 9, x ∈ bilangan ganjil}
      3. Q = {x | x < 9, x ∈ bilangan prima}
    3. P = {huruf pembentuk kata bunda}
      4. Q = {huruf pembentuk kata ibu}

  3. Diketahui himpunan-himpunan berikut. K = {x | –3 < x < 3, x ∈ bilangan bulat}; L = {lima bilangan cacah yang pertama}; M = {x | x < 5, x ∈ bilangan asli}; Dengan menyebutkan anggota-anggotanya, tentukan masing-masing anggota himpunan berikut.

    1. K ∪ L
    2. L ∪ M
    3. K ∪ M
    4. K ∪ L ∪ M

  4. Diketahui himpunan-himpunan berikut. A= {x | x < 5, x ∈ bilangan cacah}; B = {empat bilangan ganjil yang pertama}; C = {x | x < 11, x ∈ bilangan prima} Dengan menyebutkan anggota-anggotanya, tentukan masing-masing anggota himpunan berikut ini.

    1. A, B, dan C
    2. A ∪ B
    3. B ∪ C
    4. A ∪ (B ∪ C)
    5. A ∪ (B ∩ C)
    6. B ∪ (S ∩ C)
    7. C ∪ (A ∩ B)
    8. (A ∩ B) ∪ (B ∩ C)

  5. Diketahui S = {bilangan cacah kurang dari 15}; A = {x | x < 8, x ∈ S}; dan B = {x | x < 5, x ∈ S}. Nyatakan himpunan-himpunan berikut dengan mendaftar anggota-anggotanya.

    1. 3 ... A
    2. Ac
    3. Bc
    4. A ∩ Bc
    5. (A ∪ B)c
    6. A/B
    7. B/A
    8. S/A

  6. Diketahui P = {huruf pembentuk kata PERIANG}; Q = {huruf pembentuk kata GEMBIRA}; R = {huruf pembentuk kata CERIA} Dengan mendaftar anggota-anggotanya,tentukan

    1. anggota P ⋂ Q
    2. anggota Q ⋂ P
    3. anggota P ⋃ Q
    4. anggota Q ⋃ P
    5. anggota P ⋂ (Q ⋃ R)
    6. anggota P ⋃ (Q ⋂ R)
    7. anggota (P ⋂ Q) ⋃ (P ⋂ R)
    8. anggota (P ⋃ Q) ⋂ (P ⋃ R)
