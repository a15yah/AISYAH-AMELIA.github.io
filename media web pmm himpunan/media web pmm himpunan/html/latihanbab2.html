LATIHAN BAB II

  1. Di antara himpunan-himpunan berikut, tentukan manakah yang merupakan himpunan kosong?

    1. Himpunan anak kelas VII SMP yang berumur kurang dari 8 tahun
    2. Himpunan kuda yang berkaki dua
    3. Himpunan kubus yang mempunyai 12 sisi
    4. Himpunan bilangan prima yang habis dibagi 2
    5. Himpunan bilangan asli antara 8 dan 9
    6. Himpunan nama bulan dalam setahun yang berumur kurang dari 30 hari
    7. Himpunan penyelesaian untuk 2x = 3, x bilangan cacah
    8. N = {x | x + 4 = 0, x bilangan asli}

  2. Sebutkan paling sedikit dua buah himpunan semesta yang mungkin dari tiap himpunan berikut

    1. Himpunan anak kelas VII SMP yang berumur kurang dari 8 tahun
    2. G = {x | x = 2n, n bilangan cacah}
    3. H = {x | x = 2n – 1, n bilangan cacah}
    4. P = {honda, yamaha, suzuki}
    5. Q = {merpati, dara, puyuh}

  3. Tentukan sebuah himpunan semesta yang mungkin untuk himpunan-himpunan berikut

    1. A = {1, 4, 9, 16, 25}
    2. B = {1, 3, 5, 7, ... }
    3. H = {x | x = 2n – 1, n bilangan cacah}
    4. E = {m, dm, cm, mm}
    5. F = {kerucut, tabung, bola}

  4. Jika diketahui G = {faktor dari 12} dan H = {faktor dari 16}, apakah G H? Jelaskan!

  5. Jika anggota A = {1, 2, 3, 4, 5}, sebutkan himpunan-himpunan bagian dari A yang anggotanya terdiri dari

    1. A = {1, 4, 9, 16, 25}
    2. 2 anggota A
    3. 4 anggota A
    4. 3 anggota A

  6. Tentukan hubungan himpunan bagian antara himpunan-himpunan berikut!

    7. A = {2, 3, 4, 5}

    B = {bilangan asli kurang dari 7}

    C = {a, i, u, e}

    D = {huruf vokal}

    E = {a, u}

    F = {bilangan prima genap}

    G = {3,5}
